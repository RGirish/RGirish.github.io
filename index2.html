A Realtime Restaurant Management System using NFC and Cloud on Android

Project DineArt


January - May 2015


This project was my Bachelor's Degree Capstone Project. It is a Restaurant Management System, that helps automate and digitize almost all activities inside a restaurant. This system comprises of 7 applications for the Android Operating System, connected to a real-time server/databse hosted at Firebase and supported by another no-SQL database at Parse.




My Contribution

This was a project from a team of just 3. I coded all the client and server side modules, collaborating with one other team-mate. I developed the Table Module, Kitchen Module, Bearer Module and the Cashier Module completely. I designed the backend data structure at Parse and designed the control flow and logic in using the real-time server at Firebase. I also designed the User Interface for the modules that I developed.

Client-side: XML(UI Development), Java (Logic)
Server-side: Parse (no-SQL, static), Firebase (Real-time logic, temporary storage)
